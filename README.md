# SEO Optimizer

Comprehensive SEO optimization for HTML/CSS websites. Analyze, fix, and maintain perfect SEO health.

## Features

- **Automated Analysis**: Scan websites for 50+ SEO issues
- **Meta Tags**: Optimize title, description, Open Graph, Twitter Cards
- **Schema Markup**: Implement JSON-LD structured data
- **Sitemap Generation**: Create XML sitemaps automatically
- **Robots.txt**: Generate search engine access rules

## Quick Start

```bash
# Analyze a website
python scripts/seo_analyzer.py ./public

# Generate sitemap
python scripts/generate_sitemap.py ./public https://example.com
```

## Scripts

- `scripts/seo_analyzer.py` - Analyzes HTML files for SEO issues
- `scripts/generate_sitemap.py` - Generates XML sitemaps
